## Key Findings

- 핵심 조문이 명확한 질문에서는 Hybrid Search와 Reranker가 직접 근거 조문을 상위에 배치하는 경향을 보였습니다.
- 사용자 표현과 법령 표현이 다른 질문에서는 동의어 매핑 부족으로 검색 실패가 발생했습니다.
- 검색 결과가 적절해도 LLM이 조문 번호를 잘못 생성하는 문제가 있어 metadata 기반 인용이 필요합니다.
